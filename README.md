Как установить проект?
================
## Для начала работы вам необходимо:
скопировать проект:
git clone https://github.com/pavelcydep/testComment.git

## Далее следует выполнить следующюю команду:
php artisan migrate:fresh --seed

## Запуск проекта:
php artisan serve
